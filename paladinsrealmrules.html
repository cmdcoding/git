Reporting

Match wins
After each set, the captains of the winning team are to report their set score
Match reports must be reported in the format
Winning Team vs Losing Team | Map Score | Match ID and screenshot of post-game screen

Pauses

Teams are allowed 1 - 3 min pauses each game
Failure to unpause will result in automatically going to the next pause
If a team has no pauses left and fails to unpause, the team will be disqualified
Pauses can be used for player disconnects and such

Rule Breaks

If a team breaks a rule, please pause the game immediately and report it to an admin in the captains chat.
Admins may ask for proof of which rules were broken
EX: If a player uses banned skin-etc let an admin know immediately, most cases it will be a remake with same
